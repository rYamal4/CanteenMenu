## Начало работы

После запуска приложения откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

На главной странице вы увидите три основных раздела:
- **Меню** - управление блюдами
- **Категории** - управление категориями блюд
- **Ингредиенты** - управление ингредиентами

## Работа с категориями

### Просмотр категорий
1. Нажмите на "Категории" в верхнем меню
2. Вы увидите таблицу со всеми категориями
3. Для каждой категории отображается количество блюд

### Добавление новой категории
1. На странице категорий нажмите "Добавить категорию"
2. Заполните форму:
   - **Название категории** - обязательное поле (например: "Первые блюда")
   - **Описание** - необязательное поле
3. Нажмите "Сохранить"

### Редактирование категории
1. На странице категорий найдите нужную категорию
2. Нажмите кнопку "Редактировать"
3. Внесите изменения
4. Нажмите "Сохранить изменения"

### Удаление категории
1. На странице категорий найдите категорию
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление

**Важно:** Нельзя удалить категорию, если в ней есть блюда. Сначала удалите или переместите все блюда.

## Работа с ингредиентами

### Просмотр ингредиентов
1. Нажмите на "Ингредиенты" в верхнем меню
2. Таблица показывает все ингредиенты
3. Видно, в скольких блюдах используется каждый ингредиент

### Добавление ингредиента
1. Нажмите "Добавить ингредиент"
2. Заполните форму:
   - **Название** - обязательное поле (например: "Картофель")
   - **Единица измерения** - выберите из списка (г, мл, шт, л, кг)
3. Нажмите "Сохранить"

### Редактирование ингредиента
1. Найдите ингредиент в списке
2. Нажмите "Редактировать"
3. Внесите изменения
4. Сохраните

### Удаление ингредиента
1. Найдите ингредиент
2. Нажмите "Удалить"
3. Подтвердите действие

**Важно:** Нельзя удалить ингредиент, если он используется в блюдах.

## Работа с блюдами

### Просмотр меню
1. Нажмите "Меню" в верхнем меню
2. Блюда отображаются в виде карточек
3. Для каждого блюда показано:
   - Название
   - Категория
   - Описание
   - Цена
   - Вес
   - Доступность

### Фильтрация по категориям
1. На странице меню используйте выпадающий список "Категория"
2. Выберите нужную категорию
3. Список блюд обновится автоматически
4. Для сброса фильтра нажмите "Сбросить"

### Добавление нового блюда
1. Нажмите "Добавить блюдо"
2. Заполните основную информацию:
   - **Название блюда** - обязательно
   - **Категория** - выберите из списка (обязательно)
   - **Описание** - необязательно
   - **Цена** - обязательно (в рублях)
   - **Вес** - необязательно (в граммах)
   - **Доступно для заказа** - флажок (по умолчанию включен)

3. Добавление ингредиентов (необязательно):
   - Нажмите "+ Добавить ингредиент"
   - Выберите ингредиент из списка
   - Укажите количество
   - Для добавления еще ингредиентов повторите
   - Для удаления ингредиента нажмите "Удалить" рядом с ним

4. Нажмите "Сохранить"

### Просмотр деталей блюда
1. На странице меню найдите нужное блюдо
2. Нажмите "Подробнее"
3. Откроется страница с полной информацией:
   - Все характеристики блюда
   - Полный состав (список ингредиентов с количествами)

### Редактирование блюда
1. Найдите блюдо в меню
2. Нажмите "Редактировать"
3. Измените нужные данные:
   - Основную информацию
   - Состав (ингредиенты)
4. Нажмите "Сохранить изменения"

### Удаление блюда
1. Найдите блюдо
2. Нажмите "Удалить"
3. Подтвердите удаление

**Важно:** Удаление блюда необратимо!

## Рекомендуемый порядок работы

При первом использовании системы рекомендуется:

1. **Создать категории** (Первые блюда, Вторые блюда, Салаты, Напитки и т.д.)
2. **Добавить ингредиенты** (Картофель, Мясо, Овощи и т.д.)
3. **Добавить блюда** с привязкой к категориям и ингредиентам

## Советы по использованию

### Категории
- Создавайте логичные категории (Завтраки, Обеды, Ужины)
- Используйте описания для уточнения назначения категории

### Ингредиенты
- Указывайте правильные единицы измерения
- Используйте единообразные названия

### Блюда
- Указывайте подробные описания
- Указывайте точный вес для правильного учета
- Используйте флаг "Доступно" для временно недоступных блюд
- Добавляйте ингредиенты для полного контроля состава

## Клавиатурные сокращения

- `Tab` - переход между полями формы
- `Enter` - отправка формы (когда фокус на кнопке)
- `Esc` - закрытие модальных окон (если используются)

## Устранение неполадок

### Не отображаются блюда
- Проверьте фильтр по категориям
- Убедитесь, что блюда добавлены в базу данных

### Ошибка при добавлении
- Проверьте, заполнены ли обязательные поля
- Убедитесь, что категория/ингредиент существует

### Не удаляется категория/ингредиент
- Проверьте, не используется ли элемент в блюдах
- Сначала удалите все связанные блюда
